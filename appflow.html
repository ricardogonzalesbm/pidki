Sign-Up Flow

Account Type /sign-up/ (default/index)
Title

"Account Type"

UI Elements
  • Radio card group — Borrow Money / Lend Money options
  • Each option has an icon + title + description
Buttons
  • BTN Submit (disabled until selection) → /sign-up/select-state
Select State /sign-up/select-state
Title

"Welcome! Let's get started!"

Inputs
  • INPUT State dropdown / selector
Navigation
  • Routes to sign-up flow or platform-eligible based on state
Enter Phone Number /sign-up/enter-phone-number
Title

"Enter phone number"

Inputs
  • INPUT Phone Number — placeholder: "(123) 456-7890"
Buttons
  • BTN Continue → /sign-up/create-account
  • BTN Continue with Apple
  • BTN Continue with Google
  • BTN Continue with Facebook
Links
  • LINK Login → /sign-in/
Create Account /sign-up/create-account
Title

"Create account"

Inputs
  • INPUT First name — placeholder: "First name"
  • INPUT Last name — placeholder: "Last name"
  • INPUT Email address — placeholder: "email@domain.com"
Buttons
  • BTN Continue (disabled if invalid) → /sign-up/create-password
Create Password /sign-up/create-password
Title

"Create password"

Inputs
  • INPUT Password (masked) — "Enter Password"
  • INPUT Confirm password (masked) — "Re-type password"
Validation Rules (displayed inline)
  • Min 8 characters
  • Max 80 characters
  • Must include uppercase letter
  • Must include lowercase letter
  • Must include symbol
Buttons
  • BTN Continue → sign-up mutation or reset password flow
Enter Verification Code /enter-verification-code/
Title

"Enter verification code"

Subtitle
  • "Enter the 6-digit code sent to [formatted phone number]"
Inputs
  • INPUT OTP Entry — 6 boxes, auto-submits on fill
Actions
  • LINK "Resend code" — pressable text with loading indicator
Flows after submit
  • CUSTOM_WITHOUT_SRP → Sign-in success → Dashboard
  • RESET_PASSWORD → /sign-in/enter-username-password
  • Sign-up → Profile completion
